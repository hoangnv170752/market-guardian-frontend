# Kangaroo Portal

A modern admin dashboard built with:

- **Waku**: Minimal React framework
- **React 19**: Latest React version
- **Tailwind CSS**: Utility-first styling
- **Iconify**: Icon library

## Key Features

- **Responsive Layout**: Mobile screen responsive
- **Authentication**: Sign-in/Sign-up flows
- **Dashboard**: Overview metrics and navigation
- **Modern UI**: Clean design with dark/light support

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run development server:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build
pnpm start
```

## Configuration

- Edit `tailwind.config.ts` for styling
- Modify routes in `/src/pages`
- Update sidebar navigation in `/src/components/sidebar.tsx`
